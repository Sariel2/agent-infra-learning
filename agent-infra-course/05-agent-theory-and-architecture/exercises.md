# 05 Agent 原理与架构 练习题

## 概念题
1. ReAct、Router、Plan-and-Execute 的核心区别是什么。
2. workflow 和 agent 的边界如何判断。
3. 什么时候 multi-agent 是多余复杂度。

## 应用题
1. 给一个客服知识问答 + 查工单场景，判断哪些部分适合 workflow，哪些适合 agent。
2. 为一个开放性 research 任务选择一种 pattern，并说明理由。

## 排错题
1. 一个系统采用多 agent 后效果更差，可能的工程原因有哪些。
2. 一个团队把所有场景都抽成 agent，通常会遇到什么问题。

## 设计题
1. 设计一个模式选型 checklist。
2. 设计一个展示不同 pattern 差异的最小实验接口。
