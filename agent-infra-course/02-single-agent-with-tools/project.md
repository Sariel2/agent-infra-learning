# 02 单 Agent 与 Tools项目任务

## 项目名
`support-agent`

## 项目目标
做一个支持场景 Agent，可以回答问题、查询订单状态、创建工单，并输出完整工具调用轨迹。

## 必做项
- 至少 3 个工具
- 统一 tool response schema
- Agent 能根据用户意图自动选工具
- 保留执行日志
- 对失败调用给出可读反馈

## 加分项
- 支持工具优先级或黑名单
- 支持工具 mock
- 支持工具调用统计

## 验收标准
- 至少覆盖 10 条测试或手工 case
- 至少有 3 条失败 case 被记录和分析
- 主流程可多次稳定运行
