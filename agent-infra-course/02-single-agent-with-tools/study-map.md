# 02 单 Agent 与 Tools 学习地图

[返回课程首页](../README.md) | [进入模块首页](./README.md) | [进入逐步执行](./steps.md)

本模块的目标是：02 单 Agent 与 Tools 的核心知识不只“知道能用”，而是能讲清概念、原理、代码落点、常见错误与外部出处。

## 完整学习顺序
1. 先读 [resources/01-concepts-and-principles.md](./resources/01-concepts-and-principles.md)
2. 再读 [resources/02-deep-dive.md](./resources/02-deep-dive.md)
3. 再读 [resources/03-code-reading-guide.md](./resources/03-code-reading-guide.md)
4. 再读 [resources/05-common-mistakes.md](./resources/05-common-mistakes.md)
5. 最后读 [resources/04-sources.md](./resources/04-sources.md)
6. 然后按 [steps.md](./steps.md) 去跑 examples 和 module-service
7. 最后做 [exercises.md](./exercises.md) 并用 [review.md](./review.md) 复盘

## 为什么 `04-sources.md` 放在后面
因为它现在不再只是链接列表，而是“出处讲解页”。你应该先把本教程里的讲义读懂，再回到出处看它们如何支持教程中的观点，这样不会被外部网页打断主线。

## 读不下去时怎么返回
- 如果你在概念层卡住：回到 [resources/01-concepts-and-principles.md](./resources/01-concepts-and-principles.md)
- 如果你在原理层卡住：回到 [resources/02-deep-dive.md](./resources/02-deep-dive.md)
- 如果你在代码层卡住：回到 [resources/03-code-reading-guide.md](./resources/03-code-reading-guide.md)
- 如果你怀疑自己走偏了：回到 [resources/05-common-mistakes.md](./resources/05-common-mistakes.md)
- 如果你想查出处：去 [resources/04-sources.md](./resources/04-sources.md)
