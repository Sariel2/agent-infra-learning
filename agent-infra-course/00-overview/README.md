# 00 总览与准备

这个阶段是整个课程的启动器。目标不是大量编码，而是把你的学习系统搭起来，让后面 6 周每一周都围绕同一个主项目前进，而不是零散刷概念。

## 阶段目标
- 理解 Agent Infra 的完整版图
- 配好 Python 学习与实验环境
- 确定后续主项目边界
- 建立日志、实验、复盘机制

## 你会完成什么
- 一份自己的 `agent infra 全景图`
- 一个 Python 项目骨架
- 一套学习日志模板
- 一份主项目说明

## 建议投入
- 总时长：`6-8 小时`
- 适合安排在正式 Week 1 开始前，或者挪作 Week 1 的前两天

## 完成标准
- 你能说清 Agent Infra 的 8 个核心模块
- 你能用 Python 新建并运行一个 API 项目
- 你已经定义好后续 4 周主项目的场景、边界和成功标准
