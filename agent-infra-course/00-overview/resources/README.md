# 00 总览与准备 资料导航

    先建立全局地图，再开始写任何一行 agent 代码。这个模块的资料目标不是让你读完所有文档，而是让你知道后面每个模块为什么存在。

    ## 建议顺序
    1. OpenAI Responses API
2. FastAPI
3. Pydantic Settings
4. OpenTelemetry Python

    ## 使用方式
    - 先读 `concepts-summary.md`，快速建立关键词地图。
    - 再读 `key-resources.md`，理解每份资料为什么值得看。
    - 最后按 `reading-notes.md` 的节奏去读、记、跑示例。
