# 01 Python 与 Agent 基础 Examples

    这里放的是知识点级别的可运行示例。每个目录都能单独安装依赖、运行代码并执行测试。

    ## 示例列表
    - `01-structured-output`：演示用 Pydantic 模型约束 LLM 输出形状。
- `02-provider-abstraction`：演示通过 provider 接口解耦模型实现和业务逻辑。

    ## 使用方式
    1. 先按 `steps.md` 的顺序挑一个示例运行。
    2. 先跑测试，再改代码，再重新跑测试。
    3. 把观察到的行为写进 `experiments.md`。
