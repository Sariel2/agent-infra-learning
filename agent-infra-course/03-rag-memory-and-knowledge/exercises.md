# 03 RAG、Memory 与 Knowledge 练习题

## 概念题
1. RAG 的最小链路包括哪几个步骤。
2. memory 与 retrieval 的职责有什么不同。
3. citation 为什么是工程能力而不是展示能力。

## 应用题
1. 设计一个文档 chunk 方案并说明利弊。
2. 设计一个带 citation 的问答响应模型。

## 排错题
1. 问题问得很准，但系统总回答空泛，问题可能出在哪些环节。
2. 多轮问答里模型开始混淆前文和知识库内容，说明什么。

## 设计题
1. 设计一个最小 RAG 服务接口。
2. 设计一个失败样本记录表，用于后续调优。
