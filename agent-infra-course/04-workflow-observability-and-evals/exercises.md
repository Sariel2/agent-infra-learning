# 04 Workflow、Observability 与 Evals 练习题

## 概念题
1. workflow 和自由 agent 的边界是什么。
2. trace、log、eval 三者分别解决什么问题。
3. 为什么没有固定数据集就很难做回归。

## 应用题
1. 设计一个带人工审核节点的 workflow。
2. 设计一组评测指标，覆盖质量、延迟和成本。

## 排错题
1. 系统有时候好有时候坏，但没有任何运行轨迹，你该怎么补救。
2. 你做了很多 trace 但还是不知道改哪里，问题可能是什么。

## 设计题
1. 设计一个最小 trace schema。
2. 设计一个最小 eval runner 输出格式。
