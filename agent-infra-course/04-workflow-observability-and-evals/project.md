# 04 Workflow、Observability 与 Evals项目任务

## 项目名
`production-like-agent-runtime`

## 项目目标
把前几阶段积累的能力重构成一条可编排、可观测、可评测的运行时链路。

## 必做项
- 用 workflow 描述主执行过程
- 增加 trace 和关键 metadata
- 有一组固定 eval 输入
- 有一份第一次基线结果
- 有一份失败案例归因表

## 加分项
- 人工审核节点
- 回归对比脚本
- 成本统计面板

## 验收标准
- 跑一次完整流程时能看到端到端轨迹
- 能解释某次失败来自哪个节点
- 调整后能重新跑评测并比较结果
